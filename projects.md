### My fellow Singaporeans how are you feeling?

In this project I used DistilBert to analyse comments of COVID19 announcements by Prime Minister Lee Hsien Loong. 
The dataset I used was from Kaggle which has both emotion and sentiment labels. 
The results were so fascinating that I even wrote a [post](/My-Fellow-Singaporeans-How-Are-You-Feeling) about it. 

### Deep Image Captioning With Attention

Used resnet50 and LSTM with attention to form an image captioning model. 
This achieved a BLEU4 score of 24.71 with just 5 epochs on MSCOCO 2014 with vocab of about 5k words.

### SUTD Academic Scheduler

Created an academic scheduler/timetable planner. 
It uses a genetic algorithm to plan the timetable based on many different constraints such as no two teachers can use the same classroom, 
student must have adequate breaks etc...

### Librorum

Project for Database and Big Data course.
Created a web application with various database backends (MySQL, MongoDB, Hadoop).
Objective was to create a book reviews application where large amounts of user and book data has to be retrieved quickly.

Achieved Best Automation Award for the automatic deployment of the entire stack. 
Use of AWS Cloud Formation and FlintRock enabled easy one click deployment of entire stack which resulted in best automation.
